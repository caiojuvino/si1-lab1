Sitemas de Informação 1: 15.1

Objetivos da Disciplina

O foco desta disciplina é capacitar os alunos a desenvolver técnicas de projeto de software. Aprimorar a flexibilidade e reuso de software desenvolvido usando Orientação a Objeto, como também projeto arquitetural. O ambiente das aplicações será a web. Além de conseguir elaborar aplicações robustas você também será capaz de ao fim da disciplina desenvolver sistemas web.

Conteúdos Ministrados

Princípios e fundamentos de design; projeto com OO (orientação a objetos); UML (linguagem de modelagem); herança, interfaces; frameworks e bibliotecas; padrões de design; noções de arquitetura de software.

Algumas das tecnologias utilizadas na disciplina são: Bootstrap, Javascript/JQuery, Git/github. Com o uso dessas tecnologias suas aplicações web terão aparência e funcionalidades profissionais.
